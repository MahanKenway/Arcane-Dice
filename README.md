## ⚔️ What is this?

Arcane Dice is not just another dice roller. It's a **full sensory experience** for tabletop RPG players.

Watch real polyhedral dice tumble across a physics-simulated table with Ammo.js. Hear them clatter. See them settle. Then read what the AI dungeon master has to say about your roll — because every d20 tells a story.

Built for D&D players, Pathfinder groups, solo adventurers, and anyone who misses the sound of dice hitting a wooden table at 2am.

<br/>

## ✨ Core Features

| Feature | Description |
|---------|-------------|
| 🎲 **Real Physics** | Ammo.js-powered 3D dice that tumble, collide, and settle realistically |
| 🧙 **AI Narrations** | Gemini-powered dungeon master reacts to every roll with wit and drama |
| 🎨 **Custom Themes** | Prismatic, Wooden, Gemstone, Rust, Blue-Green Metal, and more |
| 🖼️ **Dynamic Backgrounds** | Cyberpunk taverns, fantasy dungeons, cozy libraries, underdark caves |
| 🎵 **Sound Effects** | Satisfying dice-on-table clack sounds for every roll |
| 📊 **Roll History** | Track your session stats — are you blessed or cursed tonight? |
| 🎯 **All Dice Types** | d4, d6, d8, d10, d12, d20, d100 — the full polyhedral set |
| ⚡ **Advantage/Disadvantage** | Roll with advantage or disadvantage, D&D 5e style |
| 🏆 **Critical Detection** | Special celebrations for nat 20s, special shame for nat 1s |

## 🚀 Quick Start

```bash
git clone https://github.com/MahanKenway/Arcane-Dice.git
cd Arcane-Dice
npm install
```

Create a `.env` file (see `.env.example`):
```env
GEMINI_API_KEY=your_key_here
```

Run the dev server:
```bash
npm run dev
```
