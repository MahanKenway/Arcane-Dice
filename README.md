## ✨ About

**Arcane Dice** is more than just rolling a random number.

It's a fantasy-inspired dice experience designed for tabletop players, RPG fans, developers, and anyone who believes every roll should feel magical.

Whether you're casting spells, testing your luck, or building your own adventure...

**The dice decides your fate.**

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/MahanKenway/Arcane-Dice.git
```

Go to the project

```bash
cd Arcane-Dice
```

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```
