# 🎲 Arcane Dice (Physics 3D Roller)

![Arcane Dice Banner](https://images.unsplash.com/photo-1605806616949-1e87b487cb2a?q=80&w=2000&auto=format&fit=crop)

A gorgeous, physics-based 3D dice roller built for tabletop RPG enthusiasts, Dungeon Masters, and anyone who loves the satisfying clatter of polyhedral dice. Powered by React, Three.js, and `@3d-dice/dice-box`.

## ✨ Features

- **Real 3D Physics:** Watch the dice tumble, collide, and settle realistically with Ammo.js physics.
- **Custom Themes & Materials:** Choose from a wide array of beautiful materials including *Prismatic*, *Wooden*, *Gemstone*, *Dice of Rolling*, *Rust*, and more!
- **Dynamic Backgrounds & Tints:** Set the perfect mood for your campaign with curated, AI-generated backgrounds (Cyberpunk, Fantasy, Underdark, Cozy Library) and atmospheric color tints.
- **Witty Narrations:** The game master is always watching! Enjoy humorous and immersive narrations based on your rolls (Critical Fails, Epic Successes, and everything in between).
- **Roll History & Stats:** Keep track of your session with a persistent roll history and statistical insights.
- **Sound Effects:** Satisfying clack sounds as the dice hit the virtual table.

## 🛠️ Built With

- **React 18** (Vite)
- **Tailwind CSS** (for that pristine, magical UI)
- **@3d-dice/dice-box** (The core 3D engine)
- **Framer Motion** (For smooth UI transitions)

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
